# Boyd Monson - Portfolio Site

This repository powers [monsonbo2.github.io](https://monsonbo2.github.io/), my personal portfolio site.

The site focuses on business intelligence, reporting systems, public health operations, and practical data workflows. It includes case studies from federal public health grant management, workflow redesign, and technical project work using tools such as SQL/SQLite, Python, FastAPI, React, TypeScript, and Excel.

## Key Pages

- Homepage: positioning and selected work
- Experience: public health operations, reporting, forecasting, and stakeholder coordination
- Projects: case studies in data workflows, reporting systems, and technical tooling
- Resume: web version of my current resume

## Local Development

This is a Jekyll/GitHub Pages site based on the AcademicPages / Minimal Mistakes ecosystem.

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
